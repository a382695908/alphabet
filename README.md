# alphabet

欢迎来到 vividfree 的 alphabet代码仓库，这个仓库将放置 Machine Learning / Natural Language Processing / Big Data Computing 相关的代码，主要用 C++ / Python / Shell 语言开发。

vividfree的博客请访问 [这里](http://vividfree.github.io/) 。

## 目录介绍

    - src: 源代码 
      - base: 基础库
        - cpp:    C++相关的基础库
        - python: Python相关的基础库
      - util: 工具库
        - cpp:    C++相关的工具库
        - python: Python相关的工具库
        - shell:  Shell相关的工具库
